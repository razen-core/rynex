# test-app

A Rynex application.

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
test-app/
├── src/
│   ├── index.ts           # Entry point
│   ├── App.ts             # Root component
│   └── components/        # Components
├── public/
│   └── index.html         # HTML shell
├── dist/                  # Build output
└── rynex.config.js       # Configuration
```
